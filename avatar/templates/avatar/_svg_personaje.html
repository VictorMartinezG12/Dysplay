{% comment %} Personaje base del avatar de DysPlay: figura 2D simplificada tipo "blob" construida 100% en SVG (sin imágenes externas), con grupos anidados que sirven de esqueleto para animar (cabeza con pivote en el cuello, brazos con pivote en hombro y subgrupo de antebrazo con pivote en el codo). Se incluye dentro de un contenedor ancestro con la clase `avatar-visual-root` y, opcionalmente, una clase `avatar-{emocion}` (ej. `avatar-feliz`) que decide qué set de cara (`.cara-*`) se muestra. Esa clase la pone el propio template que incluye este parcial (estado inicial desde la BD) o el JS de `avatar_events.js` en tiempo real (`#avatar-container`). Las prendas/accesorios equipables (ropa, cabello, calzado, accesorios) NO viven en este SVG: siguen renderizándose como `` superpuestas por z-index, exactamente igual que antes, sobre este mismo cuerpo base. {% endcomment %}
